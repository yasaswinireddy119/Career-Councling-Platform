# CareerPath AI - Full Stack Career Counseling Platform

## Project Overview
CareerPath AI is a comprehensive platform designed to help individuals navigate their professional journeys. It integrates AI-powered recommendations, expert counseling, job boards, and interactive visualizations to provide a holistic career development experience.

## Tech Stack
- **Frontend**: React, Tailwind CSS, Lucide React, Motion, D3.js, Axios
- **Backend**: Node.js, Express.js
- **Database**: Supabase (Auth, PostgreSQL)
- **AI**: Google Gemini API (@google/genai)
- **Deployment**: Render (Backend), Netlify (Frontend)

## Features
- **AI Career Recommendations**: Personalized career paths generated by Gemini based on user profiles.
- **Counselor Matching**: Algorithm-driven matching with industry experts.
- **Interactive Career Path Visualization**: D3.js powered force-directed graphs showing career progression.
- **Job Board**: Integrated job listings with filtering.
- **Secure Authentication**: Supabase-powered user and counselor login.
- **Responsive Design**: Mobile-first approach with dark mode support.

## Database Schema (Supabase)
- `profiles`: id, user_id, full_name, education, skills[], interests[], experience
- `counselors`: id, name, expertise[], rating, location, bio, availability
- `sessions`: id, user_id, counselor_id, scheduled_at, status
- `jobs`: id, title, company, location, salary, type, description
- `resources`: id, title, category, content_url, type

## Installation Steps
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Set up environment variables in `.env`:
   - `GEMINI_API_KEY`
   - `SUPABASE_URL`
   - `SUPABASE_ANON_KEY`
   - `SUPABASE_SERVICE_ROLE_KEY`
4. Run development server: `npm run dev`.

## API Documentation
- `GET /api/health`: Check server status.
- `POST /api/ai/recommendations`: Get AI-powered career paths.
- `POST /api/counselors/match`: Find matching counselors.
- `GET /api/jobs`: Fetch job listings.
